## How to Install Sliver
###### `REF` : [Sliver wiki](https://sliver.sh/docs?name=Getting+Started)

```bash
sudo apt update
curl https://sliver.sh/install|sudo bash

systemctl status sliver # 실행 확인
systemctl start sliver # sliver 실행

sliver
```
